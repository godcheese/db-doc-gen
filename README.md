# db-doc-gen
数据库文档生成器，目前仅支持 MySQL

## 特性 Features
- [ ] 参数启动
- [ ] 重新生成数据库文档需要权限

## 开发 Develop

- Dev/Compile in golang >= 1.19.3
- Dev on Fedora >= 3.7
- Compile on CentOS >= 7.9.2009
